# taiju-peiyin

泰剧 AI 配音流水线 — 用 AI 技术替代/辅助传统人工配音，降低成本、提升效率。

## 背景

泰剧引进中国的配音流程：

```
泰剧原片 → 出字幕版 → 送广电审核 → 审核通过 → 按审核字幕配音（不可改内容） → 成片
```

AI 配音的核心约束：**必须严格按照广电审核通过的字幕进行配音，不可修改内容。**

## 技术架构

```
┌─────────────────────────────────────────────────────┐
│                   输入：审核通过的中文字幕 + 原片       │
└──────────────────────┬──────────────────────────────┘
                       ▼
              ┌────────────────┐
              │  音频预处理      │
              │  (人声/背景分离) │
              │  Demucs / UVR   │
              └───────┬────────┘
                      ▼
          ┌───────────────────────┐
          │  角色识别 & 音色建模    │
          │  (Speaker Diarization) │
          │  + 语音克隆             │
          └───────────┬───────────┘
                      ▼
          ┌───────────────────────┐
          │  逐句 TTS 语音合成     │
          │  CosyVoice / Fish Speech│
          │  按字幕时间轴对齐       │
          └───────────┬───────────┘
                      ▼
          ┌───────────────────────┐
          │  语速调整 & 情感微调    │
          │  匹配原片节奏           │
          └───────────┬───────────┘
                      ▼
          ┌───────────────────────┐
          │  混音合成              │
          │  配音 + 原片背景音乐    │
          └───────────┬───────────┘
                      ▼
              ┌──────────────┐
              │  人工质检      │
              └──────┬───────┘
                     ▼
              ┌──────────────┐
              │   成片输出     │
              └──────────────┘
```

## 核心技术栈

| 环节 | 技术方案 | 备选 |
|------|----------|------|
| 语音识别 (ASR) | Whisper large-v3 | Google Cloud STT |
| 翻译 | Claude / GPT-4o | DeepL |
| 人声分离 | Demucs v4 | UVR |
| 语音克隆 | Fish Speech | GPT-SoVITS |
| 语音合成 (TTS) | CosyVoice (阿里) | ChatTTS, MegaTTS |
| 口型同步（可选） | Wav2Lip / VideoReTalking | — |
| 音频处理 | FFmpeg, pydub | — |

## 关键挑战

- **多角色音色区分** — 每个角色需独立音色，全剧保持一致
- **情感表达** — 泰剧情感丰富（哭、笑、怒），TTS 需匹配情绪
- **时间轴匹配** — 中泰语速差异大，需智能压缩/拉伸语音
- **背景音保留** — 干净分离人声，配音后混回原始背景音乐和音效

## 成本对比

| 项目 | 传统配音 | AI 配音 |
|------|----------|---------|
| 单集 (45min) | ~5,000-15,000 | ~500-2,000（含人工质检） |
| 制作周期 | 3-5 天/集 | 0.5-1 天/集 |
